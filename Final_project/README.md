# WINC Data Analytics

My projects for WINC Academy Data Analytics with Python course.

## Projects

### CO2 Emissions Analysis (Final Project)
Analysis of global CO2 emissions using data from Our World in Data.

**Research Questions:**
1. What is the biggest predictor of CO2 emissions per capita?
2. Which countries are decreasing CO2 the most?
3. Which non-fossil fuel technology will be cheapest in the future?

**Key Findings:**
- Energy per capita is the strongest predictor (r = 0.833)
- Yemen, Estonia, Finland lead in reductions
- Solar PV will be cheapest by 2030
